# Imerly Mora — Portfolio

Personal portfolio built with TypeScript and Vite, featuring a modular component architecture and EN/ES language switch.

🔗 [Live site](https://portafolio-phi-sandy.vercel.app/)

## Stack

- TypeScript
- Vite
- CSS custom properties

## Features

- EN/ES language switch
- Modular component architecture
- Responsive design
- Downloadable CV

## Run locally

git clone https://github.com/Imerly/portafolio.git
cd portafolio
npm install
npm run dev
